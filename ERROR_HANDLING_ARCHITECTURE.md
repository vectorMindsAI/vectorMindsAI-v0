## Real-World Example

### Scenario: Research Panel Crashes

```
1. User clicks "Start Research"
2. Component throws error during render
3. Section-level error boundary catches it
4. Error logged to:
   - Browser console (dev mode)
   - Analytics (PostHog)
   - API endpoint (/api/errors/log)
5. Fallback UI shown with reload button
6. Other dashboard tabs continue working
7. User can:
   - Reload just the research section
   - Switch to another tab
   - Continue using the app
```
